# Contributing

Thanks for helping improve Taglme NFC Console!

## Development

- Install prerequisites: Go, Node.js 20+, Wails v2.
- Run: `wails dev`

## Pull requests

- Keep PRs focused and small.
- Ensure CI passes (frontend build + Go build/tests).
- Prefer adding/adjusting tests when behavior changes.

## Security

If you believe you found a security issue, please follow SECURITY.md.
